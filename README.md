# vishwa.py.project2
Pattern Generator & Number Analyzer

A beginner-friendly Python project that demonstrates loops, conditional statements, user input, menus, and pattern generation. The program allows users to generate a triangle pattern, analyze numbers within a given range, and explore the use of Python control statements such as if, else, break, continue, and pass.

📌 Features
⭐ Generates a right-angled triangle pattern using nested loops.
🔢 Identifies even and odd numbers in a user-defined range.
➕ Calculates the sum of numbers within the specified range.
🎯 Demonstrates Python control statements (if, else, break, continue, pass).
📋 Interactive menu-driven interface.
✅ Handles basic input validation.
🛠️ Technologies Used
Python 3.x

▶️ How to Run
Make sure Python 3 is installed.
Download or clone this repository.
Open a terminal in the project folder.
Run the program:
python pattern_generator.py
📖 Program Flow
Displays a welcome message.
Generates a right-angled triangle pattern.
Performs number analysis (Even/Odd and Sum).
Demonstrates control statements.
Opens a menu where users can:
Generate a new triangle pattern
Analyze another range of numbers
Exit the program
