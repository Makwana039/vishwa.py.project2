# 🎯 Pattern Generator and Number Analyzer

A beginner-friendly Python project that demonstrates the use of loops, conditional statements, menus, and control statements. The program allows users to generate star patterns, analyze numbers within a range, and interact with a simple menu-driven interface.

---

## 📌 Features

- ⭐ Generate a right-angled triangle star pattern.
- 🔢 Identify even and odd numbers in a given range.
- ➕ Calculate the sum of numbers within a specified range.
- 📋 Menu-driven program for easy navigation.
- 🔁 Demonstrates Python control statements:
  - `if`
  - `elif`
  - `else`
  - `for`
  - `while`
  - `break`
  - `continue`
  - `pass`

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Clone this repository:

```bash
git clone https://github.com/your-username/Pattern-Generator-and-Number-Analyzer.git
```

3. Navigate to the project folder:

```bash
cd Pattern-Generator-and-Number-Analyzer
```

4. Run the program:

```bash
python main.py
```

---

## 📖 Program Workflow

1. Displays a welcome message.
2. Generates a right-angled triangle pattern.
3. Performs number analysis by identifying even and odd numbers.
4. Calculates the sum of numbers in a range.
5. Demonstrates Python control statements.
6. Provides a menu to repeat operations or exit the program.
